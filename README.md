# Chad's GPT

A ChatGPT parody marketing site. Clearly a parody. Comic Sans and all.

**Live site:** [chads-gpt.com](https://chads-gpt.com)

## Files

| File | Purpose |
|------|---------|
| `index.html` | The whole site |
| `style.css` | Styles |
| `chad-hero.jpg` | Hero image |

## Deploy to GoDaddy

1. Log in to GoDaddy → **cPanel → File Manager**
2. Navigate to `public_html/`
3. Upload `index.html`, `style.css`, and `chad-hero.jpg`
4. Done — site is live at chads-gpt.com

## Local preview

Just open `index.html` in your browser. No build step needed.
